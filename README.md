# hashdisco
Simple hash discovery using regex.

# usage
Usage: ./hashdisco.sh [OPTION]...

Options:
  -h | -?   print this help
  -c        left/right context length (default: 25 characters)
  -d        enable debug output
  -p  	    path to start search in (default: /)
  -C        disable colors
  -R        disable recursive search
